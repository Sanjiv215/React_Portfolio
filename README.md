# Personal Portfolio Website

## Abstract

This project is a personal portfolio website designed to showcase skills, projects, journey, and contact details in a clean and responsive layout. The objective of the project is to create a professional online presence where visitors can quickly understand the developer's technical abilities and explore completed work.

## Features

- Responsive navigation bar
- Hero section with personal introduction
- Skills section
- Featured projects with live and code links
- Journey section
- Contact section
- Footer

## Technologies Used

- React.js
- Vite
- CSS
- React Router
- React Icons

## Outcome

The final outcome is a fast, modern, and user-friendly portfolio website. It presents project work clearly, improves personal branding, and provides an easy way for visitors to connect with the developer.
